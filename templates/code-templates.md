## Function Template

### Basic Function
```
Create a {language} function named {function_name} that {description}.

Parameters:
- {param1}: {type} - {description}
- {param2}: {type} - {description}

Return: {return_type} - {return_description}

Requirements:
- {requirement1}
- {requirement2}
- Include error handling
- Add appropriate comments
```

### Class Template
```
Create a {language} class named {class_name} that {description}.

Properties:
- {property1}: {type} - {description}
- {property2}: {type} - {description}

Methods:
- {method1}({params}): {description}
- {method2}({params}): {description}

Requirements:
- Follow {language} naming conventions
- Include constructor
- Add appropriate access modifiers
- Include documentation
```
